### 学习js   
**安装**   
`npm install zarray`   
**使用**
    
``` 
import Zarray from 'zarray'   
const arr = new Zarray();
```  
去重
```
arr.qc(Array)
return Array
```
交并差集
```
arr.jbc(arr1,arr2,type)
return Array
```
出现最多的元素
```
arr.maxItem(Array)
return {items,num}
```
最大值
 ```
 arr.max(Array) 
 return number 
 ``` 
最小值
```
arr.min(Array)
return number
```
